# Airline Reservation System (COBOL)

## 📌 Description
A simple Airline Reservation System built using COBOL in a mainframe-style environment.

## 🚀 Features
- Flight booking
- Ticket cancellation
- Seat availability tracking
- Passenger record management
- Error handling using INVALID KEY

## 🛠 Technologies Used
- COBOL
- VSAM (Indexed Files - KSDS)
- JCL (for batch execution)
- DB2 (for queries - conceptual)

## 🔄 Workflow
1. Enter Flight ID
2. Check availability
3. Book or Cancel ticket
4. Display flight details

## 🧠 Key Concepts
- READ / WRITE / REWRITE
- Indexed file handling
- Data validation
- Transaction processing
